# internship-tracker
Internship Tracker Web App 🚀

A simple yet powerful web application designed to help students manage and track their internship applications in one place.

The system allows users to add companies, roles, and application status while providing a real-time dashboard to monitor progress.

🔹 Key Features
Add internship applications (Company, Role, Status)
Track application status (Applied, Interview, Selected, Rejected)
Real-time dashboard statistics
Filter applications by status
Delete entries
Data persistence using browser LocalStorage
Clean and responsive UI design
🔹 Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
LocalStorage API
🔹 Purpose of Project

This project was built to simulate a real-world internship tracking system and strengthen frontend development skills, including DOM manipulation, state management, and UI/UX design fundamentals.
